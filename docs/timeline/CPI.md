# CPI 指标

消费者价格指数（CPI，Consumer Price Index）是一个经济指标，用于衡量一定时间内城市消费者为固定一篮子商品和服务（如食品、住房、交通、医疗、服装等）支付的价格的平均变化。它主要反映以下内容：

1. **通货膨胀或通货紧缩**：
   - CPI上升通常表示通货膨胀（物价上涨），意味着消费者的购买力下降。
   - CPI下降可能表明通货紧缩（物价下跌），可能与经济活动放缓有关。

2. **生活成本变化**：
   - CPI反映了维持一定生活水平所需的成本变化，常用于调整工资、养老金或社会福利，以确保购买力不被侵蚀。

3. **经济政策依据**：
   - 政府和中央银行（如美联储）使用CPI来评估经济状况，制定货币政策。例如，CPI高可能导致加息以抑制通胀；CPI低可能促使降息以刺激经济。

4. **购买力与消费行为**：
   - CPI变化影响消费者信心和支出习惯。例如，物价快速上涨可能导致消费者减少非必需品支出。

**具体例子**：
根据2025年8月的数据，美国CPI年度通胀率为2.5%，核心CPI（不含食品和能源）为3.2%，表明物价仍在温和上涨，但能源和食品价格波动较小。这反映了经济在稳定通胀的同时，生活成本略有增加。
